Open Sonar: A Senior Design Project Completed by Graham Christie, Isaac Fuller & Kara Sanford

Found below is the educational platform for the project, providing a strong introduction of hydrography to the client.

Modules:

1. Introduction to Ocean Mapping 2. Open Sonar System 3. Open Sonar Hardware 4. Run a Survey 5. Additional Projects

Inside each module package you'll find a comprehensive introduction to the respective topic.

1. Introduction to Ocean Mapping: An introduction to the software used within the Open Sonar platform

                                                                    including how to prepare software for work, how to navigate the 

                                                                    interface and basic operation of software.

   2. Open Sonar System: An introduction to the Hardware used within the Open Sonar platform

                                                                     including component breakdowns, operation guides and vessel use.

                                                3. Open Sonar Hardware: An introduction to data collection during survey efforts such as

                                                                              vessel monitoring and controlling, understanding what features

                                                                              are being stored and how to optimize survey with field changes

4. Run a Survey: An introduction to disseminating collected data including

                                                                                       hydrographic descriptions, what is being visualized and 

                                                                                      data exportation.                                                                                                             

                                                5. Additional Projects: Customizing the Open Sonar to your own personal needs

The Project

The Open Sonar project was idealized in 2019 with ambitions high in calibre. Project ideas and preliminary planning began soon after,

with concrete progress being seen over the summer of 2021. Coming into the academic year of 2021-2022, significant progressions were 

made in the fall. The Open Sonar System  was critically analyzed and assessed with a workable interface being present. Components
